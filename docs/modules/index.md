# Module catalog

Your agent doesn't fail because the model is dumb. It fails because the **harness** around it is thin. These nine chapters teach you to fix that — one scorecard pillar at a time.

Every module fits the 4-stage journey: **① Diagnose → ② Learn → ③ Build → ④ Prove**. The spine running through all of it is the **Harness Scorecard** — 0–100 across 5 pillars, 20 points each.

::: tip Start with a number
Don't guess where you stand. **[Score your harness → /diagnose](/diagnose)** first, then read the modules that target your weakest pillars.
:::
