# Lab 01 — Copilot prompts

## Baseline (no harness)

```text
Add highlight snippets to search results in this React app.
When the user searches, matching parts of the title and body should be visually highlighted.
Mark the task done when finished.
```

Do **not** mention tests or lint. Observe what happens.

## Harness (minimal pack installed)

```text
Follow AGENTS.md and .github/copilot-instructions.md.

1. Run bash scripts/init.sh
2. Read feature_list.json and work only on feature id "search-highlight"
3. Implement acceptance criteria
4. Run npm run lint and npm test before claiming done
5. Update PROGRESS.md with verification evidence
6. Mark feature done only if all checks pass
```

## Reflection prompts

```text
Compare my baseline and harness runs. For each failure mode, which of the five pillars would have prevented it?
```
